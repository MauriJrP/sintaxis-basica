¿Qué es Css?

CSS (Cascading Style Sheets) es lo que se denomina lenguaje de hojas de estilo en cascada y se usa para estilizar elementos escritos en un lenguaje de marcado como HTML.
CSS ayuda a los desarrolladores web a crear una apariencia uniforme en varias páginas de un sitio web. Los estilos de uso común deben definirse solo una vez en un documento CSS. Una vez que el estilo se define en una hoja de estilos en cascada, puede ser utilizado por cualquier página que haga referencia al archivo CSS.
